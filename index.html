SplitView based on backbone.js and some CSS3 flavors Tweet

SplitView is a Backbone.View sub class.
This class binds mouse events on existing HTML markup in the page.
2 CSS classes are used vsplit and hsplit.

It also uses CSS3 flex box layout and gradient.

This demo works only on decent browsers.
